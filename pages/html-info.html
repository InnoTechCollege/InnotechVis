HTML

HTML +

HTML is usually the first place people start with their journey into web development. It stands for Hyper Text Markup Language.


When you visit a website, your browser is sent an HTML file (along with some other files). The browser reads the file and shows you the content.

HTML is a markup language that allows you as a developer to populate your website with content via tags. These tags contain the things you would like to display on the webpage. Things like text, images, tables, lists, and so on.

It is important to remember the point of HTML when learning web development. HTML is just for getting the content onto the page. Not for making the website look pretty or be interactive. That is for things like CSS and JavaScript.

Tags +

Tags are what make up an HTML file. Each tag plays a specific role in the page, some are designed to show text, some show images, and some are for giving structure to the page.

Learning what the different tags show and do on your page is a great start to your programming journey. Although it isn't an introduction to logical programming topics, it is a a great way to learn about how to be precise.
